# 04. 費用対効果とCFO向け説明資料

## 1. 投資の正当性（ROI予測）
本プロジェクトは、人的リソースの最適配置による実質的なコスト削減と、意思決定スピードの向上を投資対効果として定義します。

### 定量的効果（年間ベース）
- **削減工数**: 1日あたり平均2時間の問い合わせ対応 × 20日 × 5名 ＝ 200時間/月
- **削減コスト**: 200時間 × 5,000円（人件費単価）＝ **120万円/月（年間1,440万円相当）**
- **投資回収期間**: 初期開発費を300〜500万円と想定した場合、約半年で投資回収が可能。

## 2. CFOが懸念するリスクへの対応策

| 懸念点 | 対応策 |
| :--- | :--- |
| **回答の誤り（ハルシネーション）** | RAGによる根拠データの明示と、低精度時の有人エスカレーション機能により、誤った回答が放置されるのを防ぐ。 |
| **ランニングコストの増大** | Token単位の課金であり、定型的な問い合わせに絞ることで低コスト運用を維持。 |
| **情報のセキュリティ** | 社内VPN内での実行または商用利用契約（Opt-out済み）を締結したLLMモデルのみを使用。 |

## 3. 戦略的意義
- **高付加価値業務へのシフト**: 単純回答に費やしていた時間を、中期経営計画の策定や投資効率の分析に割り当てる。
- **DXのスピーディな体現**: AI技術の実務適用を経営企画が先行して行うことで、全社的なデジタル化の強力なモデルケースとする。

## 4. 費用構成案（概算）
1. **フェーズ1（PoC）**: 100〜150万円（主要QAのみ対応、特定部署での試行）
2. **フェーズ2（本導入）**: 200〜350万円（全データ連携、UI改善、全社展開）
3. **運用保守**: 月額数万円（API利用料 + メンテナンス費）

## 5. CFOへの依頼事項
- 初期フェーズ（PoC）の実行予算の承認。
- データの利活用に関するセキュリティポリシーの確認と、経理・IT部門との連携サポート。

以上により、最小限の投資で最大限の業務改善を実現する「賢い投資」としての実行を提案いたします。
