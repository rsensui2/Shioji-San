# 02. サービス概要と機能詳細

## 1. サービスコンセプト
**「経企の知識をAIが代理出力し、専門的な判断のみを人間に残す」**

本サービスは、経営企画部門が保有する「予実管理の勘定科目体系」や「費用計上のルール（月ズレ防止等）」といった専門知識をデジタル化し、事業部からの初動対応をAIが代行する社内向けソリューションです。

## 2. 主要機能詳細

### ① プロアクティブ応答機能（一次回答）
事業部からチャットで質問（例：「PC購入費の科目は何ですか？」）が投稿されると、AIが以下のプロセスで回答します。
- **データ参照**: 過去1〜3年分の仕訳履歴、社内会計規程、FAQ集を横断検索。
- **回答生成**: LLMが自然な日本語で「消耗品費として、10万円以上の場合は固定資産登録が必要です」といった具体的なガイダンスを提示。

### ② 精度判定およびオートエスカレーション
LLMが自身の回答に対して「信頼度スコア」を算出します。
- **高信頼度**: AIがそのまま回答を完了。
- **低信頼度/複雑な案件**: 「申し訳ございませんが、正確な回答のために担当者に確認します」と回答し、**自動的に経営企画の担当グループへメンション**を飛ばします。

### ③ 学習フィードバックループ
経営企画担当者が修正・回答した内容は、新たな「正解データ」としてシステムに蓄積され、次回の回答精度向上に寄与します。

## 3. ユーザーメリット

| ユーザー層 | メリット |
| :--- | :--- |
| **事業部ユーザー** | 質問への回答を待つ時間がなくなり、予算執行のスピードが上がる。 |
| **経営企画担当者** | コピペのような回答作業から解放され、予算分析や戦略立案に集中できる。 |
| **マネジメント層** | 部門全体の業務スピードが可視化され、プロセス改善のデータが得られる。 |

## 4. 利用イメージ
1. **Slack/Teams** 等の専用チャンネルに質問を投稿。
2. **AIボット** が数秒以内に回答。
3. 必要に応じて **人間（経営企画）** が補足・承認。

このように、人間の介在を最小化しつつ、最終的な正確性を担保するハイブリッドな運用体制を実現します。
